# SleepTracker

Full-stack sleep tracking app.

## Structure

- `Backend/` — Express + MongoDB API
- `Frontend/sleeptracker/` — Vite + React + TypeScript client

## Getting started

### Backend

```bash
cd Backend
npm install
npm run dev
```

### Frontend

```bash
cd Frontend/sleeptracker
npm install
npm run dev
```
